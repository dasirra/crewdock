# library-review

Comment on a GitHub repo or open-source project.

**Category:** Link-first

**Before drafting:** Visit the repo URL. Read the README. Understand what it does, who it's for, and what makes it notable.

## Structure

```
[Title: project name + what it does in one line]

[Link to repo]

[Your comment: why it matters, what you think, who should care]
```
